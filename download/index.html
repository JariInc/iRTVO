  • 1.2.15 (Nov 16, 2013)

    • Added triggering to replay button functions
    • Fixed some remote control issues

    • Fixed crashing when XSplit was not installed
    • Removed unneccessary references from example scripts
    • Added dataorder=trackposition for ordering drivers based on their absolute track position
    • Added {liveintervalfollowed} to get delta to followed driver
    • Updated about window

    • Added: Trigger:radioOn Trigger:radioOff Trigger:fastestlap
      • To show driver speanking use dataset=radio on your overlay
    • Bugfix: When moving from 3 to 2 Monitors for testing, locations are negative in the options
    • BugFix: Ignore lines in external data that do not start with a valid iracing id
    • Bugfix: Prevent crash when Api is connected but got no data yet

  • 1.2.14 (Sep 9, 2013)

    • Slomo now respects auto apply
    • Updated about window
    • hide unused labels
    • Added hotkeys
      • Add hotkeys to buttons using hotkey=Modifier-Key syntax, for example hotkey=Shift-H
      • Available modifier are None, Alt, Ctrl, NoRepeat, Shift, Win
      • All available keys are listed at http://msdn.microsoft.com/en-us/library/system.windows.input.key.aspx
      • known issue: hot keys are applied only on program start, to change hotkeys restart of whole application is required
    • Added ability to make hidden buttons using hidden=true, this goes well with the hotkey functionality
    • Added rewind functionality to buttons, use replay=10 to rewind 10 seconds, or replay=0 to return live
    • Added slow/fast motion functionality to buttons, use playspeed=2 to play 2x speed or playspeed=-2 to play 1/2x speed. playspeed=1 will return normal play speed
    • Fixed bug where a driver's data wouldn't update and showed him/her retired
    • Fixed bug that didn't update {gaplive} correctly when leader changed
